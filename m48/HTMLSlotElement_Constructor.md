# HTMLSlotElement()

The `HTMLSlotElement()` constructor creates a new 
[HTMLSlotElement](HTMLSlotElement.md) object.

## Syntax

`var htmlSlotElement = new HTMLSlotElement(name)`

### Returns

An instance of the [HTMLSlotElement](HTMLSlotElement.md) interface.

### Parameters

<dl>
  <dt>name</dt>
  <dd>The slot name.</dd>
</dl>
