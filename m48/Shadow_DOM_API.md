# Shadow DOM API

<p class="summary">
The Shadow DOM API provides a method to combine multiple DOM trees into one
hierarchy, thus enabling better composition.
</p>

## Interfaces

<dl>
  <dt>HTMLSlotElement</dt>
  <dd>Defines the location of a slot.</dd>
</dl>
